Use scenebuilder to make gui. Drag and drop stuff.
In the highest level view (it's an AnchorPane by default), 
on the right side bar, there are three menus
Properties, Layout, Code
Click on Code
In the Controller class text field, type in the class of the controller file
(the file with the java code that has the functionality)
I think you can figure out the rest.

The Model file is just there so you can start your application.
